Notes on MLE

Jul 17, 2021 • 2 min read

statistics   mle

The process of estimating parameter $\theta$ from data $\mathcal{D}$ is called model fitting and MLE is one of the most common approach. MLE picks the parameter [Murphy 2021] that assigns highest probability to the training data.

θ^mle≜arg⁡max⁡θp(D∣θ)\hat{\theta}_{mle} \triangleq \arg\max_\theta p(\mathcal{D}|\theta)θ^mle​≜argθmax​p(D∣θ)

Under iid assumption the likelihood is given by

p(D∣θ)=∏i=1Np(yi∣xi;θ)p(\mathcal{D}|\theta) = \prod_{i=1}^N p(y_i|x_i; \theta)p(D∣θ)=i=1∏N​p(yi​∣xi​;θ)

The log likelihood is defined as

LL(θ)≜log p(D∣θ)=∑i=1Nlog p(yi∣xi;θ)\begin{aligned} LL(\theta) &\triangleq \text{log } p(\mathcal{D}|\theta)\\ &= \sum_{i=1}^N \text{log } p(y_i|x_i; \theta) \end{aligned}LL(θ)​≜log p(D∣θ)=i=1∑N​log p(yi​∣xi​;θ)​

Maximizing some function $f(\cdot)$ is same as minimizing $-f(\cdot)$. The negative log likelihood is given by

NLL(θ)≜−log p(D∣θ)=−∑i=1Nlog p(yi∣xi;θ)\begin{aligned} NLL(\theta) &\triangleq - \text{log } p(\mathcal{D}|\theta)\\ &= -\sum_{i=1}^N \text{log } p(y_i|x_i; \theta) \end{aligned}NLL(θ)​≜−log p(D∣θ)=−i=1∑N​log p(yi​∣xi​;θ)​

Minimizing NLL will give us the MLE

θ^mle=arg min⁡θ−log p(D∣θ)=arg min⁡θ−∑i=1Nlog p(yi∣xi;θ)\begin{aligned} \hat{\theta}_{mle} &= \argmin_\theta - \text{log } p(\mathcal{D}|\theta)\\ &= \argmin_\theta -\sum_{i=1}^N \text{log } p(y_i|x_i; \theta) \end{aligned}θ^mle​​=θargmin​−log p(D∣θ)=θargmin​−i=1∑N​log p(yi​∣xi​;θ)​

MLE as point estimate to Bayesian posterior

MLE can be thought as a point estimate of Bayesian posterior $p(\theta|\mathcal{D})$ under uniform prior. The model of the posterior is given by

θ^map=arg max⁡θlog p(θ∣D)=arg max⁡θlog p(D∣θ)+log p(θ)=arg max⁡θlog p(D∣θ)+const {uniform prior}=θ^mle\begin{aligned} \hat{\theta}_{map} &= \argmax_\theta \text{log } p(\theta|\mathcal{D}) \\ &= \argmax_\theta \text{log } p(\mathcal{D}|\theta) + \text{log } p(\theta) \\ &= \argmax_\theta \text{log } p(\mathcal{D}|\theta) + const \text{ \{uniform prior\}}\\ &= \hat{\theta}_{mle} \end{aligned}θ^map​​=θargmax​log p(θ∣D)=θargmax​log p(D∣θ)+log p(θ)=θargmax​log p(D∣θ)+const {uniform prior}=θ^mle​​

MLE is equivalent to minimizing KL

Computing MLE is equivalent to minimizing Kullback Leibler (KL) divergence with empirical distribution of data. The empirical distribution of data for unsupervised case is given by

pdata≜=1N∑i=1Nδ(y−yi)p_{data} \triangleq = \frac{1}{N}\sum_{i=1}^N\delta(y - y_i)pdata​≜=N1​i=1∑N​δ(y−yi​)

When estimating parameter, we want model $q(y) = p(y|\theta)$ that is similar to empirical distribution. We can measure the goodness of the fitted model by calculating distance between $p_{data}$, the empirical distribution and $q$, the model.

KL(p∣∣q)=∑yp(y) log p(y)q(y)=∑yp(y) log p(y)−∑yp(y) log q(y)=∑ypdata(y) log pdata(y)−∑ypdata(y) log q(y)=−H(pdata)−1N∑i=1N log q(y)=−H(pdata)−1N∑i=1N log p(yi∣θ)=const+NLL(θ)\begin{aligned} \mathbb{KL}(p||q) &= \sum_{y} p(y) \text{ log } \frac{p(y)}{q(y)}\\ &= \sum_{y} p(y) \text{ log } p(y) - \sum_y p(y) \text{ log } q(y)\\ &= \sum_{y} p_{data}(y) \text{ log } p_{data}(y) - \sum_y p_{data}(y) \text{ log } q(y)\\ &= -\mathbb{H}(p_{data}) - \frac{1}{N}\sum_{i=1}^N\text{ log } q(y)\\ &= -\mathbb{H}(p_{data}) - \frac{1}{N}\sum_{i=1}^N\text{ log } p(y_i|\theta)\\ &= const + NLL(\theta) \end{aligned}KL(p∣∣q)​=y∑​p(y) log q(y)p(y)​=y∑​p(y) log p(y)−y∑​p(y) log q(y)=y∑​pdata​(y) log pdata​(y)−y∑​pdata​(y) log q(y)=−H(pdata​)−N1​i=1∑N​ log q(y)=−H(pdata​)−N1​i=1∑N​ log p(yi​∣θ)=const+NLL(θ)​

Examples

Bernoulli distribution

Let $\theta = p(y=head)$ then NLL of this distribution is given by

NLL(θ)=−∑i=1Nlog p(yi;θ)=−∑i=1Nlog (θI(yi=1)⋅(1−θ)I(yi=0))=−∑i=1NI(yi=1)log(θ)+I(yi=0)log(1−θ)=−[Nheadlog(θ)+Ntaillog(1−θ)]\begin{aligned} NLL(\theta) &= -\sum_{i=1}^{N}\text{log } p(y_i; \theta) \\ &= -\sum_{i=1}^{N}\text{log }\big(\theta^{\mathbb{I}(y_i = 1)}\cdot(1 - \theta)^{\mathbb{I}(y_i = 0)}\big)\\ &= -\sum_{i=1}^{N} \mathbb{I}(y_i = 1) log(\theta) + \mathbb{I}(y_i = 0) log(1 - \theta)\\ &= -\big[N_{head} log(\theta) + N_{tail} log(1 - \theta)\big] \end{aligned}NLL(θ)​=−i=1∑N​log p(yi​;θ)=−i=1∑N​log (θI(yi​=1)⋅(1−θ)I(yi​=0))=−i=1∑N​I(yi​=1)log(θ)+I(yi​=0)log(1−θ)=−[Nhead​log(θ)+Ntail​log(1−θ)]​

MLE can be found by solving $\frac{d}{d\theta} NLL(\theta) = 0$

ddθNLL(θ)=−Nheadθ+Ntail(1−θ)=0\frac{d}{d\theta} NLL(\theta) = -\frac{N_{head}}{\theta} + \frac{N_{tail}}{(1 - \theta)} = 0dθd​NLL(θ)=−θNhead​​+(1−θ)Ntail​​=0

Solving for $\theta$ gives

θ^mle=NheadNhead+Ntail\hat{\theta}_{mle} = \frac{N_{head}}{N_{head} + N_{tail}}θ^mle​=Nhead​+Ntail​Nhead​​

Categorical distrbution

Categorical distribution is generalization of bernouli to $k$-category. The NLL is given by

NLL(θ)=−∑kNklog(θk)NLL(\theta) = -\sum_{k}N_{k}log(\theta_k)NLL(θ)=−k∑​Nk​log(θk​)

To compute MLE we have to minimize NLL with contraint $\sum_k^K \theta_k = 1$. Using lagrangian

L(θ,λ)≜−∑kNklog(θk)−λ(1−∑kKθk)∂L∂λ=(1−∑kKθk)=0∂L∂θk=−Nkθk+λ=0  ⟹  Nk=λθk\begin{aligned} \mathcal{L}(\theta, \lambda) &\triangleq -\sum_{k}N_{k}log(\theta_k) -\lambda\Big(1 -\sum_k^K \theta_k \Big)\\ \frac{\partial\mathcal{L}}{\partial\lambda} &= \Big(1 -\sum_k^K \theta_k\Big)=0\\ \frac{\partial\mathcal{L}}{\partial\theta_k} &= -\frac{N_k}{\theta_k}+ \lambda=0 \implies N_k = \lambda\theta_k \end{aligned}L(θ,λ)∂λ∂L​∂θk​∂L​​≜−k∑​Nk​log(θk​)−λ(1−k∑K​θk​)=(1−k∑K​θk​)=0=−θk​Nk​​+λ=0⟹Nk​=λθk​​

Using sum to one contraint we have

∑kNk=N∑kλθk=Nλ∑kθk=Nλ=N\begin{aligned} \sum_k N_k &= N\\ \sum_k \lambda\theta_k &= N\\ \lambda \sum_k\theta_k &= N \\ \lambda &= N \end{aligned}k∑​Nk​k∑​λθk​λk∑​θk​λ​=N=N=N=N​

Thus MLE is given by

θk^=Nkλ=NkN\hat{\theta_k} = \frac{N_k}{\lambda} = \frac{N_k}{N}θk​^​=λNk​​=NNk​​
